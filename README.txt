Este proyecto corresponde a mi Trabajo Fin de Máster (TFM) y tiene como objetivo predecir el abandono o la graduación de estudiantes universitarios a partir de variables académicas, financieras y sociodemográficas, utilizando modelos de Machine Learning.

Link del Dataset en Kaggle --> https://www.kaggle.com/datasets/naveenkumar20bps1137/predict-students-dropout-and-academic-success

📂 Archivos principales

app.py → Código de la aplicación en Streamlit, que permite predecir la probabilidad de abandono o graduación de un estudiante. Para ejecutarlo necesitas ejecutar "streamlit run app.py". También puedes acceder con este link:
🔗 Streamlit App: https://tfm-predicci-n-abandono-universitario-kmdsukt8tbqmtwsrf8mappb.streamlit.app/ 
⚠️ Nota: Si aparece como "Dormida", simplemente haz clic en Despertar app y en unos segundos estará disponible.

modelo_logistic.pkl → Modelo entrenado de Regresión Logística, necesario para que funcione app.py.

df_students.csv → Dataset traducido al español, usado por la aplicación para generar predicciones.

TFM - Memoria - Alejandro Carbonero.pdf → Documento completo de la memoria del TFM. Aquí realizo un informe basándome en la información que considero mas importante y tratando de no incorporar mucho dato técnico.

Código - TFM - Alejandro Carbonero -> Esto incluye todo el código ejecutable con comentario acerca de todos los pasos que sigo en la elaboración del modelo. La versión utilizada es: Python 3.11.11
